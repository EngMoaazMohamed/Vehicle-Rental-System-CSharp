# Vehicle Rental System — C#

A console-based vehicle rental management system built in C# as a project for Object-Oriented Programming course.

## Classes
- **Vehicle** — Abstract base class
- **Car, SUV, Truck, Motorcycle** — Derived vehicle types
- **Customer** — Manages customer data and loyalty discounts
- **Rental** — Handles rental transactions and receipts
- **RentalManager** — Core engine managing fleet, customers, and rentals

## OOP Concepts Used
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Interfaces

## Features
- Rent and return vehicles
- Automatic pricing with type-based surcharges
- Loyalty discount system (5% after 3 rentals, 10% after 5)
- Rental receipts with cost breakdown
- Revenue summary dashboard
